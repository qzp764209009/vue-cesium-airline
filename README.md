# vue-cesium-airline 🚀
  
**vue-cesium-airline** 是一个基于 [vue3+vite+cesium] 的 [应用]，用于 [无人机航点航线设计]。

## 主要功能 ✨
- 新增航点
- 新增视锥体
- 视锥体范围展示
- 偏航角控制
- 俯仰角控制
- 焦距控制

## Prerequisites 📋
运行该项目前，请确保已安装：
- [Node.js](https://nodejs.org/) (v20+)

## Installation ⚙️
克隆仓库并安装依赖：
```bash
git clone https://github.com/qzp764209009/vue-cesium-airline.git
pnpm install 
pnpm run dev
